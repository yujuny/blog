spring mvc 多数据源配置（一）：根据package配置不同的数据源

发表于 2018-07-18 | 分类于 spring mvc

背景

有时候项目中可以需要连接两个数据库。因此就需要用到多数据源配置。

原理就是配置两个dataSource，两个entityManagerFactory分别使用不同的dataSource，并设置不同的package扫描路径。

applicationContext.xml文件
