spring mvc 异步调用 @Async

发表于 2018-07-20 | 分类于 spring mvc

定义

“异步调用”对应的是“同步调用”，同步调用指程序按照定义顺序依次执行，每一行程序都必须等待上一行程序执行完成之后才能执行；异步调用指程序在顺序执行时，不等待异步调用的语句返回结果就执行后面的程序。

调用方法和异步函数不能在一个 class 中。
