背景

kettle 拥有很多自带的组件，能帮我们实现很多的功能。

但是我们总有一些很复（qi）杂（pa）的需求，用自带的组件实现不了，或者说实现起来很复杂。那么这时我们就要用到万能的组件了（ Java 代码），通过自己写代码来实现任何想要的功能。
